---
title: TagInput
description: Multi-value tag input for comma-separated or enter-separated tag entry.
---

## Import

```tsx
import { TagInput } from '@dangbt/pro-ui'
```

## Basic usage

```tsx
<TagInput label="Tags" placeholder="Add tags..." />
```

## Controlled

```tsx
const [tags, setTags] = useState(['react', 'typescript'])

<TagInput
  label="Technologies"
  value={tags}
  onChange={setTags}
  placeholder="Add technology..."
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `label` | `string` | ‚Äî | Label |
| `value` | `string[]` | ‚Äî | Controlled tags |
| `defaultValue` | `string[]` | ‚Äî | Uncontrolled default |
| `onChange` | `(tags: string[]) => void` | ‚Äî | Called on change |
| `placeholder` | `string` | ‚Äî | Placeholder |
| `maxTags` | `number` | ‚Äî | Maximum number of tags |
