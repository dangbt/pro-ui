# Pro Tier Roadmap — @dangbt/pro-ui

> **Pro components** will be available as `@dangbt/pro-ui-pro` — a separate npm package with a license key, available to [GitHub Sponsors ($15+/mo)](https://github.com/sponsors/dangbt) and via one-time purchase.

---

## Current: Free (MIT) — @dangbt/pro-ui

Everything that's already shipped and always free:

- ✅ ProTable — server-side data table with search, sort, pagination, selection, bulk actions
- ✅ ProForm — schema-driven forms with Zod validation
- ✅ Layout — sidebar + horizontal top navigation (3 variants + sub-nav)
- ✅ 30+ standard components (Button, Modal, Drawer, Tabs, Badge, Alert, Statistic, Steps, Toast, …)
- ✅ Dark/Light/System theme
- ✅ MCP Server (mcp-pro-ui) — AI-native component discovery
- ✅ CLI scaffolder (create-pro-ui-app)
